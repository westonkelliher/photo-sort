photo-sort

Stop staring at 12,000 unsorted phone photos. One command files ten years of camera roll by capture date.

Try it in your browser Get it on GitHub Sponsor

What it does

photo-sort ~/Pictures/dump -d ~/Pictures/sorted -c --dedupe

Reads each photo's EXIF DateTimeOriginal. Falls back to filename patterns. Falls back to file mtime. Files everything into YYYY/MM-Month/. SHA-1 dedupe is a flag. Dry-run is a flag. Nothing leaves your computer.

Why

  • No cloud upload. Your photos stay yours.
  • No subscription. One Python file, MIT licensed.
  • Works on JPG, PNG, HEIC, TIFF, WEBP, MP4, MOV, AVI, MKV.
  • Tells you which date source it used per file so you can audit.
